# Tile & Character Art — Attribution

## Sprout Lands — Basic Pack (RETIRED, no longer used)

The game **no longer uses** the Sprout Lands pack by Cup Nooble. All terrain,
characters, buildings, and props now render from owned PixelLab-generated art
(`assets/tiles/pixellab/`, `assets/chars/`, `assets/objects/`) plus a couple of
procedurally generated tiles (`assets/tiles/owned/`, see `tools/make_tiles.mjs`).
This was done so the game can be deployed publicly without redistributing a
third-party asset pack.

Historical note: earlier versions vendored the free Sprout Lands pack under
`assets/tiles/sprout/` (git-ignored). That directory is no longer referenced by
any code and can be deleted. Original pack:
- Source: https://cupnooble.itch.io/sprout-lands-asset-pack
- Author: Cup Nooble

## PixelLab — generated art (owned)

The pixel art under `assets/tiles/pixellab/`, `assets/chars/`, and
`assets/objects/` was generated with PixelLab (pixellab.ai) for this project.

## Procedural tiles (owned)

`assets/tiles/owned/` holds small tileable textures generated by
`tools/make_tiles.mjs` (no external dependencies).
