# Herb Plate Images — Attribution

Botanical plate images used in the identify (E-dialog) and Kräuterbuch.
The UI looks for `assets/plates/<herb.plate>` — if the file is missing the
pixel sprite is shown instead (graceful fallback in both `book.js` and
`identify.js`).

## License

All images are **public domain** (publication before 1927).  Primary source:
**Köhler's Medizinal-Pflanzen**, Franz Eugen Köhler, 1887 (vol. 1) / 1890
(vol. 2) / 1898 (vol. 3). Colour lithographs by W. Müller & C.F. Schmidt.
Fallback source: **Flora von Deutschland, Österreich und der Schweiz**,
Otto Wilhelm Thomé, 1885–1905 (coloured plates). Both series are hosted on
Wikimedia Commons and confirmed public domain.
