# Herb Plate Images — Attribution

Botanical plate images shown in the identify (E) dialog and the Kräuterbuch.
The UI looks for `assets/plates/<herb.plate>`; if the file is missing the herb's
ground sprite is shown instead (graceful fallback in `book.js` and `identify.js`).

## License

All images are **public domain** (published before 1927). Sources: **Köhler's
Medizinal-Pflanzen** (1887–1898), **Thomé, Flora von Deutschland, Österreich und
der Schweiz** (1885–1905), **Sturm, Deutschlands Flora**, **Flora Batava**, and
related works, all hosted on Wikimedia Commons. Re-fetched 2026-06-14.
