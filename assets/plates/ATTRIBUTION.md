# Herb Plate Images — Attribution

Botanical plate images used in the Kräuterbuch go here as `<speciesId>.png`
(e.g. `loewenzahn.png`, `brennnessel.png`).

The book UI looks for `assets/plates/<herb.plate>` as a CSS background-image.
If the file is missing the plate area stays empty — the pixel sprite is always
shown regardless, so missing plates degrade gracefully.

## Sourcing plates

Public-domain botanical illustrations are available from:

- **Wikimedia Commons** — search for the Latin name + "Sturm" or "Köhler" or
  "Flora von Deutschland"
- **Biodiversity Heritage Library** (biodiversitylibrary.org) — large archive
  of public-domain botanical works, downloadable as high-res TIFF/JPG
